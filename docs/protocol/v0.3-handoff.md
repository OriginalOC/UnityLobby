# UnityLink v0.3 Handoff

This document captures the planned scope for UnityLink v0.3 in the UnitySTEAM direction.

Primary targets:
- Install/update transactions (atomic, resumable)
- UnityIEC proxy/cache protocol (object presence, prefetch, upstream)
- Observability (stats endpoints, structured error detail codes)
- Minimal auth/identity evolution hooks (session tokens, user_id)

Non-goals for v0.3:
- UDP transport
- delta/patch objects (beyond hooks)
- DRM/encryption
- rollback netcode
- full accounts/OAuth
